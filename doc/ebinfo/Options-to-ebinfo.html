ebinfo Options

The ebinfo command supports both traditional single-letter options and mnemonic long option names. Long option names are indicated with -- instead of -. Abbreviations for option names are allowed as long as they are unique.

-h
--help
Output help message to standard out, then exit.
-l
--book-list
Suppress normal output; instead output a list of books that an EBNET server provides. book-path command line argument must be a remote access identifer.
-m
--multi-search
Also output information about multi search.
-v
--version
Output version number to standard out, then exit.
