Book list on an EBNET server

Instead of usual output, ebinfo can output a list of books that an EBNET server provides. Summary of its command line usage is: 

% ebinfo --book-list remote-server-identifier

where remote-server-identifier must be a remote access identifier without book name. For example, 

% ebinfo --book-list ebnet://dict.example.com
Name             Title
encycl           Brown Concise Encyclopedia
encycl.app       Brown Concise Encyclopedia (appendix)
crossword        Crossword Puzzle Dictionary
travel           World Travel Guide

The server has three books and one appendix. Then, you can see information about the book encycl by this way: 

% ebinfo ebnet://dict.example.com/cncyclopedia
disc type: EB/EBG/EBXA/EBXA-C/S-EBXA
character code: ISO 8859-1
the number of subbooks: 1
subbook 1:
  title: Brown Concise Encyclopedia
  directory: brown
  search methods: word endword keyword multi menu
  font sizes: 16 24
  narrow font characters: 0xa121 -- 0xa452
  wide font characters: 0xa321 -- 0xa452
