Summary of ebstopcode Options

The ebstopcode command supports both traditional single-letter options and mnemonic long option names. Long option names are indicated with -- instead of -. Abbreviations for option names are allowed as long as they are unique.

-c code
--code code
Specify code as text stop code (e.g. "0x1f09 0x0001"). ebstopcode displays text until code is appeared.
-h
--help
Output help message to standard out, then exit.
-l length
--text-length length
Outputs text data length bytes maximum. If length is set to 0, ebstopcode outputs the whole text. By default, the length is set to 2048.
-n
--no-candidates
Suppress stop code candidates on output.
-p page:offset
--text-position page:offset
Start outputting text data at this position. page must be a hexadecimal integer greater than 0, and offset must be a hexadecimal integer between 0 and 0x7FF. If the option is not specified, ebstopcode starts outputting at the beginning of text data.
-v
--version
Output version number to standard out, then exit.

--code (-c) and --length (-l) options override each other.
